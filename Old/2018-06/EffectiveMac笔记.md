## Emacs快捷键

1. Ctrl + A：移动到行首
2. Ctrl + E：移动到行尾
3. Ctrl + K：删除到行尾
4. Ctrl + N：移动到下一行
5. Ctrl + P：移动到上一行
6. Ctrl + F：向右（Forward）移动一个字母，等价于方向键 →
7. Ctrl + B：向左（Backward）移动一个字母，等价于方向键 ←
8. Ctrl + D：向右删除一个字母，等价于 → + Delete这个快捷键也很常用
9. Ctrl + H：向左删除一个字母，等价于 Delete
10. Option + ←：光标向左移动一个单词
11. Option + →：光标向右移动一个单词
12. Option + Delete：删除一个单词
