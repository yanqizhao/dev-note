ctrl + n 下一行  
ctrl + p 上一行  
ctrl + f / b 左右  
ctrl + alt + f / b 左右单词  
ctrl + d / h 删除  
ctrl + a / e 行首 行尾  
ctrl + k 删除一行  
ctrl + cmd + ↑ / ↓ .h .m切换  
ctrl + cmd + ← / → 文件前后跳转  
cmd + shift + o 搜索文件  
cmd + 6 显示函数定义  
ctrl + 5 显示文件夹中的文件  
cmd + shift + , edit scheme  
cmd + , 设置
